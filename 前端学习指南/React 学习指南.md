这篇文章收集了学习 React 的各种精品学习资源，旨在帮助初学者快速入门和实践 React 。

2018 年 React 学习路径图：[https://zhuanlan.zhihu.com/p/39744174](https://zhuanlan.zhihu.com/p/39744174)

这份学习资源将会分成以下几个层次，学习者可以按照这个层次逐步学习：

- 了解 React
- 学习 React
- 实践 React
- 深入理解 React

每个层次里面的文章顺序也是按照学习的顺序排列，初学者只要按照这个文档的链接顺序进行学习。

## 了解 React

[图解 React](https://juejin.im/post/5b481f6b51882519ad6175c2)

作者从设计师的角度出发，帮助你用图形化的方式了解 React 是什么。

[What is React](https://learnreact.design/2017/06/08/what-is-react/)

这个博客是英文版的。作者旨在传播即使是设计师也能看得懂的 React 知识。React，React Native，Components，Props，State，React Native vs Ionic etc...

[React 官网教程](https://reactjs.org/tutorial/tutorial.html) 英文版
[React 官网教程](https://doc.react-china.org/tutorial/tutorial.html) 中文版

React 官方出品，通过一个教程，快速的了解 React 是如何工作的。

## 学习 React

[React 入门实例教程](http://www.ruanyifeng.com/blog/2015/03/react.html)

阮一峰出品的，通过 11 个 React 实例，帮助你快速建立起对 React 最基本和最核心概念的认知。

[React 官方文档](https://reactjs.org/docs/getting-started.html) 英文版
[React 官方文档](https://doc.react-china.org/docs/hello-world.html)   中文版

React 官方出品，一步一步通过实例的方式，帮助你了解 React 最基本和最核心的概念认知。

## 深入理解 React

[React 技术栈教程](http://www.ruanyifeng.com/blog/2016/09/react-technology-stack.html)

阮一峰出品的，React 全家桶的实例教程，帮助你了解一个完整的 React 开发流程需要哪些东西，如何学习这些东西。

[React in patterns](https://github.com/SangKa/react-in-patterns-cn)

Github 上 [React in patterns](https://github.com/krasimir/react-in-patterns) 的翻译版。在学习完基础 React 知识之后，能够让你对 React 的整体以及其生态圈有一个整体的认识。

[react-bits](https://github.com/vasanthk/react-bits)

Github 上总结的关于 React 的一份全面设计模式指南和最佳实践。

[react-howto](https://github.com/petehunt/react-howto/blob/master/README-zh.md)

Facebook 前工程师，React 的维护者之一写得一份 “如何学习 React 的指南”

[awesome-react](https://github.com/enaqx/awesome-react)

react 的 awesome 系列，罗列了大多数 React 生态圈的链接。
